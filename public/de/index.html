What is this community?

A couple guys got together one day and asked each other “What do you think about a place that IT folk can get help from others in the local area?” The idea sounded great, and this place was born.

  • Contacts

    Right now the best contact for this community is the Github repo, until the forum can be created.

  • This site

    This site is created with Hugo. The code is hosted on Github, and the webserver is Cloudflare Pages.

  • Support

    There may be a need to have some monetary support for this community as it becomes bigger. Right now, donations are not needed

  • What's coming

    A forum, possibly a matrix server, maybe a knowledge base. Anything is possible
